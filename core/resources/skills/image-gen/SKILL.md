---
name: image-gen
description: Enables image generation tools when user requests visual content
allowed_tools:
  - image-generate
  - image-edit
---

Use this skill when the user wants you to create or edit images.

After loading this skill, use image-generate to create new visuals from text and image-edit to revise existing images.

Clarify the goal, style, and constraints before generating. Prefer iterative edits when the user wants adjustments to an existing result.
