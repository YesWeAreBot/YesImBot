    片段的生命周期状态

    • open: 开放中，正在接收新事件
    • closed: 已关闭，通常在 Agent 介入时发生，等待系统进一步处理
    • folded: 已折叠，其关联的 AgentTurn 因历史过长被从上下文中移除
    • summarized: 已总结，原始内容已被LLM压缩成摘要
    • archived: 已归档，记录在被物理删除前的最终状态，不参与上下文构建
