Enable this skill when the user needs current information, fact-checking, or details from specific web pages.

Use search first to gather relevant sources, then use fetch on high-value URLs to read full page content.

Typical triggers include recent news, time-sensitive facts, release status checks, and verification of claims that may be outdated in model-only knowledge.
