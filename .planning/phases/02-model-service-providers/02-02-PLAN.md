Create `providers/provider-openai/src/index.ts` as a Koishi plugin:

**Exports:**
- `name = 'yesimbot-provider-openai'`
- `inject = ['model-service']` (depends on ModelService from core)
- `Config` schema with:
  - `instanceName: Schema.string().required().description('Unique instance name')`
  - `apiKey: Schema.string().role('secret').required()`
  - `baseURL: Schema.string().default('https://api.openai.com/v1')`
  - `models: Schema.array(Schema.object({ id: Schema.string().required(), capabilities: Schema.array(Schema.string()) })).default([{ id: 'gpt-4o', capabilities: ['toolCalling', 'vision', 'jsonMode', 'streaming'] }])`
  - `defaultParams: Schema.object({ temperature: Schema.number(), maxTokens: Schema.number(), topP: Schema.number() }).default({})`

**OpenAIProvider class** implementing IModelProvider:
- Constructor takes Config, creates `createOpenAI({ apiKey, baseURL })` client
- `instanceName` and `providerType = 'openai'` from config
- `models`: map config.models to ModelInfo[] (convert string capabilities to ModelCapability enum)
- `getModel(modelId)`: return `this.client(modelId)` — createOpenAI returns a callable that produces LanguageModel
- `getDefaultParams(modelId)`: return config.defaultParams

**apply function:**
- Create OpenAIProvider instance from config
- `ctx['model-service'].registerProvider(config.instanceName, provider)`
- `ctx.on('dispose', () => ctx['model-service'].unregisterProvider(config.instanceName))`
- Log registration via `ctx.logger('provider-openai')`
