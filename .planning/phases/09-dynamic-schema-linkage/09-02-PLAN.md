In `plugins/core/src/services/agent/loop.ts`:

   ```ts
   function parseModelId(fullId: string): { provider: string; model: string } | null {
     const idx = fullId.indexOf(':');
     if (idx < 0) return null;
     return { provider: fullId.slice(0, idx), model: fullId.slice(idx + 1) };
   }
   ```
   Place this as a module-level function at the bottom of the file.

2. In `run()`, replace the current provider/model resolution:
   - Old: `if (!config.provider || !config.model) { ... }` then `modelService.getModel(config.provider, config.model)`
   - New: Parse `config.model` (which is now `provider:model` string). If parse fails or config.model is empty, warn and return.
   - Use parsed provider/model for `modelService.getModel(parsed.provider, parsed.model)`
   - For `modelService.streamCall` and the `generateText` direct call, use `parsed.provider` and `parsed.model`

3. Add fallback logic: if the primary model's provider is not found in modelService, try `config.fallbackModel` with the same parse logic. Log a warning: "Primary model {config.model} unavailable, falling back to {config.fallbackModel}". If fallback also fails, warn and return.
