  <action>
Create `willingness.ts` as a plain class (NOT a Koishi Service) with:

1. **CooldownState interface**: `{ lastReplyAt: number; messagesSinceReply: number }`
2. **WillingnessCalculator class** with `private cooldowns = new Map<string, CooldownState>()` and these methods:

- `async shouldReply(percept: UserMessagePercept, config: AgentCoreConfig, modelService: ModelService): Promise<boolean>`:
  - Build channel key from `percept.scope.platform:percept.scope.channelId`
  - If `percept.triggerType === "mention" || percept.triggerType === "reply"` return `true` (deterministic bypass per user decision)
  - If `isInHardCooldown(key, config)` return `false`
  - Compute rule score via `computeScore(percept, key, config)`
  - If score < `config.willingnessRejectThreshold ?? 0.15` return `false`
  - If score >= `config.willingnessAcceptThreshold ?? 0.75` return `true`
  - Otherwise call `llmJudge(percept, config, modelService)`

- `private computeScore(percept: UserMessagePercept, key: string, config: AgentCoreConfig): number`:
  - Trigger weights: direct=0.9, keyword=0.6, random=0.2 (mention/reply already handled)
  - Soft decay: if cooldown state exists, `decayFactor = Math.min(1, elapsed / (config.willingSoftDecayMs ?? 300000))`, multiply score by `0.5 + 0.5 * decayFactor`
  - Return clamped 0-1

- `private isInHardCooldown(key: string, config: AgentCoreConfig): boolean`:
  - If no state, return false
  - `msgOk = state.messagesSinceReply >= (config.willingCooldownMessages ?? 3)`
  - `timeOk = Date.now() - state.lastReplyAt >= (config.willingCooldownMs ?? 60000)`
  - Return `!(msgOk && timeOk)` — in cooldown if EITHER condition not met (per user decision: "取较长者")

- `private async llmJudge(percept: UserMessagePercept, config: AgentCoreConfig, modelService: ModelService): Promise<boolean>`:
  - Use `config.willingnessProvider ?? config.provider` and `config.willingnessModel ?? config.model`
  - If neither available, return `false` (fail-safe)
  - Call `modelService.getModel(provider, model)` then `generateText({ model, ...defaultParams, system: "You decide if a chatbot should reply to this message. Answer only 'yes' or 'no'.", prompt: "Message: \"${percept.payload.content}\"\nTrigger: ${percept.triggerType}\nShould the bot reply?", maxTokens: 5 })`
  - Return `text.trim().toLowerCase().startsWith("yes")`
  - Wrap in try/catch, return `false` on error (fail-safe: don't reply on LLM error)

- `recordReply(key: string): void` — sets `cooldowns.set(key, { lastReplyAt: Date.now(), messagesSinceReply: 0 })`
- `incrementMessageCount(key: string): void` — increments `messagesSinceReply` if state exists

Then extend `AgentCoreConfig` in `config.ts` with these fields (flat, willingness-prefixed per research recommendation):

- `willingnessProvider?: string`
- `willingnessModel?: string`
- `willingnessRejectThreshold?: number`
- `willingnessAcceptThreshold?: number`
- `willingCooldownMessages?: number`
- `willingCooldownMs?: number`
- `willingSoftDecayMs?: number`

Update `index.ts` to re-export `WillingnessCalculator`.

Import `generateText` from "ai" and types from horizon/types and model/service using `import type` where possible.
