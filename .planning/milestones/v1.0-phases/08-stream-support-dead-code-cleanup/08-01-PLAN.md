The queue slot is released when `streamText()` returns the result object (stream is lazy — HTTP connection established, not fully consumed). Pattern matches `call()` method. The `handleStreamFallback` call should also be inside the queue callback so fallback attempts respect concurrency.

```

  try {
    // existing provider lookup + streamText call
  } catch (error) {
    // existing error classification + handleStreamFallback
```

Add `archiveStale(scope: Scope, olderThanMs: number): Promise<void>` to `EventManager`. It bulk-updates all entries matching `{ scope, stage: TimelineStage.Active, timestamp: { $lte: cutoff } }` to `{ stage: TimelineStage.Archived }`. Use the same `as unknown as Query.Expr<TimelineEntry>` cast pattern from `query()`. Compute cutoff as `new Date(Date.now() - olderThanMs)`.

Add `archiveThresholdMs?: number` to `HorizonServiceConfig`. Default 86400000 (24 hours).

**index.ts — Schema + config pass-through:**
